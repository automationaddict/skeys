# Display Settings

## Overview

The Display tab allows you to customize the visual appearance of SKeys for better readability and comfort.

## Text Size

Adjust the text size throughout the application. This affects all text elements including labels, buttons, and content.

### Available Sizes

- **Small** (85%): Compact view, fits more content on screen
- **Normal** (100%): Default size, balanced readability
- **Large** (115%): Easier to read, recommended for larger displays
- **Extra Large** (130%): Maximum readability, ideal for accessibility needs

### Preview

As you select different text sizes, a preview shows how text will appear at that size. Changes take effect immediately throughout the app.

## Tips

- If you find yourself squinting or leaning closer to the screen, try a larger text size
- On high-resolution displays, larger text sizes often improve comfort
- The setting is saved automatically and persists between sessions
