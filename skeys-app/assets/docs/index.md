# Welcome to skeys

skeys is a comprehensive SSH key and server management application that simplifies working with SSH connections, keys, and configurations.

## Quick Start

New to skeys? Here's what you can do:

- **[SSH Keys](skeys://help/keys)** - Generate, manage, and organize your SSH keys
- **[SSH Agent](skeys://help/agent)** - Add keys to the agent for passwordless authentication
- **[SSH Server Status](skeys://help/server)** - Monitor and control your SSH server
- **[Configuration](skeys://help/config-client)** - Manage SSH client and server configurations
- **[Remote Servers](skeys://help/remotes)** - Connect to and manage remote SSH servers

## Learn About SSH

If you're new to SSH or want to understand the concepts better:

## Features

### SSH Key Management
Generate and manage SSH keys with support for multiple key types (RSA, Ed25519, ECDSA). View key fingerprints, copy public keys, and organize your keys in one place.

### SSH Agent
Add keys to the SSH agent for passwordless authentication. Monitor which keys are loaded and manage key lifetimes.

### Server Control
Start, stop, and monitor your SSH server. Configure server settings and view connection status.

### Configuration Management
Edit SSH client and server configurations with a user-friendly interface. Manage host entries, aliases, and connection options.

### Host Management
Keep track of known hosts and authorized keys. Verify host key fingerprints and manage access control.

### Backup & Restore
Backup your SSH configurations and keys. Restore from backups to quickly set up on new systems.
